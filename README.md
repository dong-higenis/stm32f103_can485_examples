1. toggle_led
 - LED(GPIOB.14)와 LED(GPIOB.15) 토글

2. button_led
 - SW4(GPIOB.12) 를 누르면 LED(GPIOB.14) 점등, 떼면 소등
 - SW5(GPIOB.13) 를 누르면 LED(GPIOB.15) 점등, 떼면 소등

3. uart1_printf
 - RS232 포트(UART1) 에서 Hello World -  출력

4. rs485_send
 - RS485 포트(UART3) 에서 Hello World 출력
